If you’d like to contribute to Apache Crail, please use the follwing resources.

Source code

The versioned source code history is available from the Apache git repository at http://git-wip-us.apache.org/repos/asf/incubator-crail.git or from the github mirror https://github.com/apache/incubator-crail

How to contribute to Crail

For any potential changes/proposals we recommend that you open a JIRA ticket to have a disucssion. After making necessary code changes, please open a pull request at Github, and let us know on the dev mailing list.

Bug/JIRA tracking

https://issues.apache.org/jira/browse/CRAIL

Mailing list

Please join the Crail developer mailing list for discussions and notifications. The list is at: dev@crail.incubator.apache.org.
